# Draw-Run
Тестовое задание Draw&amp;Run.  
  
Задание:
Нужно повторить кор-механику этой игры.  
(https://play.google.com/store/apps/details?id=com.butchersgames.drawandrun)  
  
Нужно сделать:  
✔️ Панель рисования/управления построением бегунов;  
✔️ Анимацию бегунов, их движение и смерть;  
✔️ Добавление новых бегунов;  
✔️ Препятствия и смерть от них;  
✔️ 1 уровень на свое усмотрение;  
✔️ При построении дорожек(трассы) нужно использовать Dreamteck сплайны.
